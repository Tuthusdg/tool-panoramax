# 📦 Panoramax – Toolbox

Welcome to the Panoramax toolbox, a collection of scripts to make working with 360° panoramic images easier.

## 🛠️ Installation

To install the toolbox, run the following commands in your terminal:

```bash
git clone https://github.com/Tuthusdg/tool-panoramax/
cd tool-panoramax
source setup.sh
python boite_a_outils.py

The setup.sh script will:

    Create a virtual environment named tool_panoramax

    Automatically install the required dependencies via pip

🔄 Restarting the tool later

To restart the toolbox later, follow these steps:

Go to the repository folder:

cd ~/tool-panoramax/

Activate the virtual environment:

source tool_panoramax/bin/activate

Launch the main script:

python boite_a_outils.py

⚠️ Prerequisites

Before installation, make sure you have:

Python 3.8 or later

Git installed on your machine

bash (to run the setup.sh script)
